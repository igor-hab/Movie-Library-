# Movie Library

**Автор:** [Мельник Игорь]

**Дата:** 04.05.2026

## Описание

Приложение для ведения личной кинотеки с графическим интерфейсом. Позволяет:
* Добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга.
* Фильтровать записи по жанру и году выпуска.
* Сохранять и загружать данные в формате JSON.

## Как запустить приложение

**Требования:**
* Python 3.6 или выше;
* библиотека Tkinter (обычно идёт в комплекте с Python).

**Запуск:**
1. Откройте терминал или командную строку.
2. Перейдите в директорию проекта: `cd path/to/MovieLibrary`.
3. Запустите приложение: `python main.py`.

## Примеры использования

**Тест 1: Добавление фильма**
1. Заполните поля:
   * **Название:** «Интерстеллар»;
   * **Жанр:** выберите «Фантастика»;
   * **Год выпуска:** 2014;
   * **Рейтинг:** 8.6.
2. Нажмите **«Добавить фильм»**.
3. Убедитесь, что запись появилась в таблице.

## Особенности реализации

* **Валидация данных:** программа проверяет заполнение всех полей, корректность года выпуска (1888–2100) и рейтинга (0–10).
* **Сохранение между запусками:** все фильмы сохраняются в `movies.json`.
* **Гибкая фильтрация:** можно фильтровать одновременно по жанру и минимальному году выпуска.
* **Интуитивный интерфейс:** все элементы управления расположены логично и понятно.
